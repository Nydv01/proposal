# Media assets (replace samples with your files)

Drop your real files here — paths match `content.json`.

| Folder | Format | Example |
|--------|--------|---------|
| `images/` | `.webp`, `.jpg`, `.png` | `autumn_cafe.webp` |
| `videos/` | `.mp4` (H.264, muted loop) | `first_meeting.mp4` |
| `audio/` | `.mp3` or `.ogg` | `our_song.mp3` |

Until you add files, the site uses **cinematic procedural** video & generative music automatically.
